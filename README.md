# 🎓 Academic Peer Connector

A modern web-based student collaboration platform that helps students connect with peers for academic support, skill sharing, and project collaboration.

---

## 📌 Project Overview

Academic Peer Connector is designed to bridge the gap between students by providing a platform where they can:

- Connect with students from different departments
- Share academic skills and knowledge
- Send and accept collaboration requests
- Exchange learning resources
- Track notifications and requests
- Build academic connections within the institution

This project was developed using PHP, MySQL, HTML, CSS, JavaScript, and Bootstrap.

## 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/Suganilavan/academic-peer-connector.git
```

### 2. Move the project

Copy the project folder into:

```
C:\xampp\htdocs\
```

### 3. Start XAMPP

Start:

- Apache
- MySQL

### 4. Create Database

Open:

```
http://localhost/phpmyadmin
```

Create a database named:

```
academic_peer_connector
```

Import the SQL file.

### 5. Run the Project

Open your browser:

```
http://localhost/AcademicPeerConnector
```
